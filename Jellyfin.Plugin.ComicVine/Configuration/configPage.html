An override for issue name formats. Leave blank for default behavior.
Available Placeholders:
  • {Name} - Name of the issue. Blank if no name exists.
  • {Volume.Name} - The volume name of the comic. Blank if no name exists.
  • {Id} - The comic vine ID of the comic.
  • {IssueNumber} - The raw issue number of the comic. (Example: 1, 5, 10)
  • {IssueNumberFormatted} - The formatted issue number of the comic. (Example: 001, 005, 010).
